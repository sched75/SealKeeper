Contact

Three dedicated channels exist for different needs. Choose the one that matches your message. All addresses are routed via Cloudflare Email Routing to the project maintainer.

Security vulnerabilities

For any security vulnerability, suspected exploit, cryptographic weakness, or other issue that could compromise the confidentiality or integrity of SealKeeper deployments, please use the following address. PGP-encrypted reports are preferred and will be acknowledged within 72 hours.

Address. security@sealkeeper.eu

PGP fingerprint. 1E11 670E 11A0 4C0D 808F  B94D CCC4 93CB A701 6D6E — see the Security policy for the responsible disclosure timeline, and verify the key against keys.openpgp.org or /.well-known/pgp-key.asc before encrypting sensitive content.

Code of Conduct reports

For reporting violations of the Code of Conduct — harassment, discriminatory behaviour, or any conduct that is incompatible with the spirit of an inclusive open-source community — please use the following address.

Address. conduct@sealkeeper.eu

Reports are handled by the project maintainer with confidentiality. The reporter's identity is not shared with the subject of the report without explicit consent.

Code of Conduct escalation

If your concern involves the project maintainer themselves, or if you feel that a Code of Conduct report has not been handled appropriately, please use the escalation address. This channel is monitored by a trusted third party (to be confirmed before the v0.1.0 release) and bypasses the regular maintainer.

Address. conduct-escalation@sealkeeper.eu

Use only for escalations. For first reports, see the conduct address above.

General inquiries and discussions

For all other matters — questions about the project's direction, integration help, partnership inquiries, press requests, anything not covered above — please open a discussion on the project's GitHub:

github.com/sched75/sealkeeper/discussions

GitHub Discussions provides better thread tracking than email and keeps the conversation accessible to the wider community.
